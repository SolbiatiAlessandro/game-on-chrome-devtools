Install Game On Mac

Download this game on your computer

Right click on the folder, services -> new terminal at folder

Write in the terminal `python -m SimpleHTTPServer`

Open the game at 0.0.0.0:8000

CMD + SHIFT + I -> Sources -> Filesystem -> Add Folder To Workspace, and select downloaded folder

Click allow and you can play the game!
