Install Game On Mac

1) Download this game at https://github.com/SolbiatiAlessandro/game-on-chrome-devtools/archive/refs/heads/master.zip

2) Unzip it, right click on the folder, and services -> new terminal at folder

3) Write in the terminal `python -m SimpleHTTPServer`

4) Open the game at http://0.0.0.0:8000

5) CMD + SHIFT + I -> Sources -> Filesystem -> Add Folder To Workspace, and select downloaded folder

6) Click allow and you can play the game!
